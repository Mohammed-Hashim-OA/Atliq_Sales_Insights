📊 Sales Insights Dashboard – Power BI Project
🚀 Project Overview

This project is an interactive Sales Insights Dashboard developed using Power BI to analyze and visualize the sales performance of Atliq Hardware across different years, markets, and customer segments.

The goal of this project was to transform raw sales data into meaningful business insights that help in understanding revenue trends, sales performance, and customer behavior through interactive visualizations and KPI tracking.
